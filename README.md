# TATETI en C++

Este es un pequeño juego de Tateti (Tres en línea) desarrollado íntegramente en C++. 

El proyecto fue creado como práctica de programación, utilizando:
* Lógica de matrices para el tablero.
* Funciones y prototipos** para organizar el código.
* ZinjaI como entorno de desarrollo (archivo .zpr incluido).

Cómo jugar
1. Descarga o clona el repositorio.
2. Abre el archivo `TATETI.zpr` en ZinjaI.
3. Compila y ejecuta para jugar contra un amigo en la misma consola.

Tic-Tac-Toe in C++
This is a simple Tic-Tac-Toe game developed entirely in C++.

The project was created as a programming practice, featuring:

Matrix logic for the game board.

Functions and prototypes for structured code organization.

ZinjaI as the IDE (includes the .zpr project file).

How to Play
Download or clone the repository.

Open the TATETI.zpr file in ZinjaI.

Compile and run to play against a friend in the console.
