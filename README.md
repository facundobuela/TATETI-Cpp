# TATETI en C++

Este es un pequeño juego de Tateti (Tres en línea) desarrollado íntegramente en C++. 

El proyecto fue creado como práctica de programación, utilizando:
* Lógica de matrices para el tablero.
* Funciones y prototipos** para organizar el código.
* ZinjaI como entorno de desarrollo (archivo .zpr incluido).

Cómo jugar
1. Descarga o clona el repositorio.
2. Abre el archivo `TATETI.zpr` en ZinjaI.
3. Compila y ejecuta para jugar contra un amigo en la misma consola.
